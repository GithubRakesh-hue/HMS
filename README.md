# 🏥 Hospital Management System

A simple **Hospital Management System** built using **Python** and **CSV File Handling**. This project allows users to manage patient records, perform hospital administration tasks, generate reports, and analyze patient billing data through a menu-driven interface.

---

## 🚀 Features

### 👨‍⚕️ Patient Management

* ➕ Add New Patient
* 📋 View All Patients
* 🔍 Search Patient by ID
* ✏️ Update Patient Details
* ❌ Delete Patient Record
* 🏥 Discharge Patient

### 💰 Billing Management

* 🧾 Display Patient Bill
* 📈 Calculate Highest Bill
* 📊 Calculate Average Bill
* 💵 Total Revenue Calculation

### 📑 Reports & Analytics

* 👨‍⚕️ Doctor-wise Patient Report
* 🦠 Disease-wise Patient Report
* 🔢 Total Patient Count
* 📋 Complete Summary Report
* 📅 Daily Management Report

### 🏢 Hospital Information

* ℹ️ Hospital Profile
* 👨‍⚕️ Doctor-wise Patient Count

## 📊 Patient Record Format

The system stores patient information in a CSV file using the following fields:

| Field          | Description               |
| -------------- | ------------------------- |
| Patient ID     | Unique Patient Identifier |
| Name           | Patient Name              |
| Age            | Patient Age               |
| Gender         | Male / Female             |
| Disease        | Disease Name              |
| Doctor Name    | Assigned Doctor           |
| Admission Date | YYYY-MM-DD                |
| Bill Amount    | Total Bill                |
| Status         | Admitted / Discharged     |
